# 投资分析方法(多头 / 空头 / 估值 / 护城河)

本文件整理基本面投资分析的常用框架,供分析"看多(Bull)"与"看空(Bear)"观点时检索参考。

## 多头观点(Bull Case)
看多一只股票的核心论据通常包括:
- **营收与利润高增长**:营收同比增速、利润率扩张。
- **市场空间大(TAM)**:所处赛道总可达市场规模大,增长可见性强。
- **竞争护城河(Moat)**:技术壁垒、品牌、网络效应、规模成本优势、切换成本等。
- **催化剂(Catalysts)**:新品发布、行业政策利好、订单/指引上调、并购整合等。

## 空头观点(Bear Case)
看空或做空的论据通常包括:
- **估值过高(Overvaluation)**:市盈率/市销率远高于历史或同业,已计入乐观预期。
- **增长放缓**:营收增速见顶、市场份额流失。
- **利润率压力**:成本上升、价格战、监管限制。
- **财务质量隐患**:现金流与利润不匹配、应收/库存异常堆积。
- **周期与集中度风险**:客户/产品高度集中,周期反转时业绩剧烈波动。

## 估值方法(Valuation)
- **相对估值**:用 P/E(市盈率)、P/S(市销率)、EV/EBITDA 等与同行或历史比较。
- **绝对估值**:DCF(现金流折现),将未来自由现金流折现求和。
- 估值要结合增速与确定性:高增长可支撑更高估值倍数。

## 护城河(Moat / Competitive Advantage)
持久的竞争优势来源:
- 无形资产(品牌、专利、牌照)
- 转换成本(用户迁移成本高)
- 网络效应(用户越多价值越大)
- 成本优势(规模效应、工艺)
- 高效规模(自然垄断/寡头)

## 风险管理
- **仓位与分散**:避免单一个股/单一行业过度集中。
- **催化剂与时间**:明确看多/看空的触发条件与兑现节奏。
- **证伪信号**:预先设定什么数据出现就推翻原假设(如增速跌破某阈值)。

## 分析报告常见结构
1. 执行摘要(Executive Summary) 2. 关键财务指标 3. 市场与竞争分析 4. 增长/风险驱动 5. 投资建议与估值 6. 风险因素。每项结论都应**引用来源与数据**,避免无依据的推断。
