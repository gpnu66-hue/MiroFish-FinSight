# 财报结构与阅读要点(Earnings Report / 10-K / 10-Q)

上市公司定期披露财务报告。理解其结构,有助于准确提取数据、避免被噪音误导。

## 报告类型
- **10-K**:美国上市公司年度报告,经审计,信息最完整。
- **10-Q**:季度报告,未经审计(或有限审阅)。
- **财报新闻稿(Earnings Release)**:业绩发布时的简报,含关键指标与管理层评论。
- **电话会(Earnings Call)**:管理层解读业绩并回答分析师提问,常包含未来**指引(Guidance)**。

## 三大财务报表
1. **利润表(Income Statement)**:营收 → 成本 → 毛利 → 营业利润 → 净利润。看增长与利润率。
2. **资产负债表(Balance Sheet)**:资产 = 负债 + 股东权益。看资本结构、现金、债务。
3. **现金流量表(Cash Flow Statement)**:经营/投资/筹资现金流。**自由现金流(FCF)** = 经营现金流 − 资本开支。

## 阅读要点
- **同比 vs 环比**:用 YoY 剔除季节性判断真实趋势。
- **分部披露(Segment)**:按业务板块看营收与利润,识别真正的增长引擎(如某公司的"服务业务"或"数据中心")。
- **毛利率/营业利润率变化**:判断是规模效应还是成本压力。
- **指引(Guidance)**:管理层对下季度/全年的预测,往往是股价催化剂。
- **股票回购与股本变化**:流通股减少会推高 EPS,需与净利润增速对比看"真实"增长。

## 常见陷阱
- 只看净利润不看现金流:利润可能含一次性项目或应收堆积。
- 忽略一次性项目(One-off):资产出售、税收优惠等会扭曲同比。
- 把"指引"当"实际":guidance 是预测,不代表已实现。
- 单季度波动当趋势:至少看 4 个季度(年度同比)再下结论。

## 数据引用规范
分析结论应标注来源,例如:营收 $XXX billion(来源:公司 FY20XX 10-K / 财报新闻稿)。对预测性数据应明确标注为"指引/预测"而非已实现。
